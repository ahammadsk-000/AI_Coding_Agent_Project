# Slim runtime dependencies for the free-tier deploy (Render, 512 MB).

# managed deployment actually needs. It deliberately EXCLUDES the heavy /
# host-only packages so the app fits a 512 MB free dyno:

#                              cross-encoder reranker is skipped gracefully.

#                              managed PaaS does not provide (SANDBOX_ENABLED=false).
#   - flower                 → Celery monitoring UI; no worker runs here
#                              (ingestion runs inline — INGEST_INLINE=true).
#
# Keep these pins in sync with pyproject.toml.
